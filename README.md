# VetCare - Guía de Configuración y Ejecución

Guía rápida para instalar, abrir y ejecutar el proyecto en tu entorno local.

---

## Requisitos e Instalación

### 1. Abrir el proyecto
Abre la carpeta del proyecto en tu editor de código preferido (por ejemplo, VS Code) y asegúrate de situar tu terminal en el directorio raíz del proyecto.

### 2. Instalar dependencias
Instala los paquetes y dependencias del proyecto ejecutando el siguiente comando en la terminal:
```bash
npm install
```

---

## Ejecución del Proyecto

### 1. Iniciar el servidor de desarrollo
Para compilar la aplicación y levantar el servidor local en modo de desarrollo, ejecuta:
```bash
npm start
```

### 2. Abrir en el navegador
Una vez que la compilación finalice con éxito, abre tu navegador e ingresa a la siguiente dirección:
[http://localhost:4200/](http://localhost:4200/)
